# Flight Fight
this is the official github repo for the best VR flying-FPS game in the whole entire world

## How to test it for yourself
1. clone the repo
2. export it somewhere safe
3. open godot
4. import the project file
5. learn how to export to apk or _(better option)_ stream the game using PCVR

OR

1. download the latest apk from the releases tab
